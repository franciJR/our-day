# Akhil & Sharlet Wedding Website

A beautiful, elegant wedding website celebrating our special day on April 18, 2026.

## Features

- Elegant hero image banner
- Ceremonies timeline with all event details
- Love story timeline
- Floating photo animations
- Fully responsive design
- Ivory and dark grey color theme

## Deployment

This website can be easily deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

Simply upload all files including the `photos` folder to your hosting service.
