INSTRUCTIONS FOR ADDING YOUR PHOTOS:

1. Add your photos to this folder with these exact names:
   - photo1.jpg (or .png)
   - photo2.jpg (or .png)
   - photo3.jpg (or .png)
   - photo4.jpg (or .png)
   - photo5.jpg (or .png)
   - photo6.jpg (or .png)

2. Supported formats: .jpg, .jpeg, .png, .webp

3. For best results, use images that are at least 800x600 pixels

4. You can rename your photos to match these names, or update the file names in index.html to match your photo names

5. If you have more or fewer than 6 photos, you can add/remove photo-item divs in the index.html file
